Image drop-in folder.

The deck renders styled placeholders for every photo. To replace a
placeholder with the real photo, save a file in this folder using the
exact filename listed below. The renderer will pick it up automatically.

Recommended sizing:
  Single photo slides:  3840 x 2160 (16:9), JPG quality 80 or higher
  VC grid tiles:        1600 x 1200 each
The theme applies a navy scrim and slight desaturation, so use full-color
source photos. Lime never appears inside a photo, that rule is applied
by the theme.
